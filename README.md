📁 Smart File Organizer PRO
A modern Python GUI application to automatically organize files by type.
---
 Features
- Select any folder easily
- Automatically organize files into folders
- Supports Images (.jpg, .png)
- Supports PDFs (.pdf)
- Supports Text files (.txt)
- Progress bar with real-time status
- Clean and modern user interface
---
Tech Stack
- Python
- PyQt5
- OS module
- shutil module
---
How to Run

1.Clone the repository:
git clone https://github.com/Vaibhavi564/Smart-File-Organizer.git

2.Navigate to project folder:
cd Smart-File-Organizer

3.Run the application:
python main.py

---
Create EXE File
python -m PyInstaller --onefile --windowed main.py
